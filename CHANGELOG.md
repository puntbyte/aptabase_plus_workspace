## 0.1.0

- Split the original Flutter-only SDK into a pure Dart core and a Flutter integration package.
- Added `aptabase_storage_shared_preferences_plus` as a dedicated `SharedPreferencesAsync` storage adapter.
- Added `aptabase_storage_hive_ce_plus` as a dedicated Hive CE storage adapter.
- Removed the direct `shared_preferences` dependency from `aptabase_flutter_plus`.
- Added package-local examples and mirrored test directories.
- Rewrote workspace and package READMEs with architecture, usage, migration, and testing guidance.
